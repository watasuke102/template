# プロジェクトテンプレート

## なんぞこれ
プロジェクトテンプレートです。コピペして開発をすぐ開始できます。

## 何があるの
- OpenSiv3D
- Qt
- DXライブラリ（Windows専用）
- React

## 使い方
OpenSiv3D/Qtは各ライブラリをインストールし、CMakeLists.txtやc_cpp_properties.jsonに書いてあるパスを修正したら基本的に使えます

その他はフォルダに有るREADME.mdを参照してください
